Faster, Cheaper and Better

All Entity Resolution does is identifying the information belonging to particular entity from disparate sources and merge them into single record. This problem was identified in 1950 and Dunn named it as Record Linkage problem even before this concept was automated in computers. There are alternative jargon used to explain this concept like Data Matching, Deduplication etc.
